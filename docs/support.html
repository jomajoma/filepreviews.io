We're always happy to help with code or other questions you might have. There are a few channels for getting support:

  • Email - support@getblimp.com
  • Live Chat - Blimp Support Chat Room
  • Twitter - @Blimp

If you're looking for support on a specific library, check the documentation specific to that client. Several clients are maintained by the Blimp team, whereas others are not. Most libraries have their own bug tracker, which is generally where the code is hosted (i.e. on GitHub).
